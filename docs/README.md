# React 19 Crash Course — Full Documentation

Welcome! This is a complete, beginner-friendly course on **React 19**. Every section has its own folder with a `README.md` (the lesson) and an `examples/` folder (real code you can copy and run).

We use **simple English** and a **Problem → Solution** style: first we show the problem, then we solve it with React.

---

## How to use these docs

1. Start at Section 1 and go in order — each section builds on the last.
2. Read the lesson (`README.md`), then open the files in `examples/`.
3. Type the code yourself. Don't just read it.
4. Build the two projects (Sections 13 and 14).
